# TJU_TCP：TCP协议在应用层的复现与优化
> 天津大学计算机网络实践
> Score: 95+

## 简介

本实验的主要目标为在应用层实现TCP协议，包括建立连接、可靠数据传输、流量控制、关闭连接与拥塞控制。通过使用C语言编写相关程序，通过网络仿真测试，能够实现TCP协议的全部功能。

在协议设计及实现方面，通过阅读实验指导书，结合已经给出的代码框架，对发送方和接收方两端的socket接口函数进行了模拟，从而实现各模块功能。该项目在完成全部任务后，能够模拟三次握手、数据传输和四次挥手等全部情况，实现了TCP的基本功能。
